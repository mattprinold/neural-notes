# Neural Notes

An agentic AI platform for contextual memory, meeting intelligence, semantic retrieval, and persistent knowledge workflows.

## Overview

Neural Notes is a production-style AI engineering project focused on building a "second brain" system capable of understanding, storing, retrieving, and reasoning over contextual information from meetings, notes, and knowledge sources.

The platform combines:
- Retrieval-Augmented Generation (RAG)
- Persistent semantic memory
- Agentic AI workflows
- Evaluation-driven development
- Observability and tracing

to explore modern patterns in AI systems engineering.
